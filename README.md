# Pokémon Chatbot Project

## Overview

This project is a Pokémon-themed chatbot developed in Python. The chatbot provides information about Generation 1 Pokémon, including their types, strengths, and weaknesses. It can also compare two Pokémon and predict which one has a battle advantage based on type matchups.

The project was created to practice chatbot development, file handling, user interaction, and basic Natural Language Processing concepts.

## Features

* Display Pokémon type information
* Show strengths and weaknesses
* Compare two Pokémon using battle predictions
* Read questions from an input file
* Generate results in an output file
* Interactive chatbot mode

## Tools Used

* Python
* Regular Expressions (re)
* Dictionaries
* File Input/Output
* String Processing

## Skills Demonstrated

* Chatbot Development
* Python Programming
* Data Organization
* User Interaction Design
* Rule-Based AI Systems
* Input Validation
